# FamilyCare TV: Roku Installation Guide

## Requirements
1. A physical Roku Device (Roku Express, Streaming Stick, Ultra, or Roku TV)
2. A computer on the same Wi-Fi network as the Roku device.
3. The `FamilyCareTV_Roku.zip` channel package.

## Step 1: Enable Developer Mode on Roku
1. Using the Roku remote, press the following sequence:
   **Home 3x, Up 2x, Right, Left, Right, Left, Right**
2. You will see the "Developer Settings" screen. Note the IP address displayed (e.g., `http://192.168.1.50`).
3. Select "Enable installer and restart".
4. Agree to the Developer Tools License Agreement.
5. Enter a password when prompted (remember this password!).
6. The Roku device will restart.

## Step 2: Sideload the Channel
1. On your computer, open a web browser and enter the IP address of your Roku device.
2. Log in using the username `rokudev` and the password you set in Step 1.
3. You will see the Roku Development Application Installer page.
4. Click **Upload** and select the `FamilyCareTV_Roku.zip` file.
5. Click **Install**.
6. If successful, the FamilyCare TV channel will automatically launch on your Roku device.

## Step 3: Configure Roku Screensaver
To enable the FamilyCare TV screensaver:
1. Press **Home** on the Roku remote.
2. Go to **Settings > Theme > Screensavers**.
3. Select **FamilyCare TV Screensaver**.
4. Set it as the active screensaver.
5. Go to **Settings > Theme > Screensaver start time** and adjust it to your preference (e.g., 5 minutes).
